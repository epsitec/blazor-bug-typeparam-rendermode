# Minimal project used to demonstrate Blazor issue

- Issue description: [dotnet/aspnetcore 54864](https://github.com/dotnet/aspnetcore/issues/54864)

## Problem

The component `Demo.razor` with this content:

```razor
@rendermode InteractiveServer
@typeparam TItem where TItem : IFormattable

<h3>Demo</h3>
```

does not compile, because the generated code is incorrect.

The attribute:

```cs
[global::BlazorApp1.Components.Demo.__PrivateComponentRenderModeAttribute]
```

should be written as:

```cs
[global::BlazorApp1.Components.Demo<TItem>.__PrivateComponentRenderModeAttribute]
```
